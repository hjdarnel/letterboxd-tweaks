# Letterboxd Tweaks

A set of userscripts to add quality-of-life improvements to [Letterboxd](https://letterboxd.com/)

## Features

### Sitewide Hotkeys
| Key | Action |
|-----|--------|
| `/` | Open search and focus the search input |
| `Esc` | Close search (when search input is focused) |

### Film Page Hotkeys
| Key | Action |
|-----|--------|
| `W` | Toggle watchlist status (add/remove) |

## Installation

### Prerequisites
- A userscript manager: [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/)
